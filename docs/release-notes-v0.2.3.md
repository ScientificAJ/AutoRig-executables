# AutoRig Executables v0.2.3

## Highlights

- Added film facial eyelid color payload support:
  - `options.film_extension.facial_plugin.appearance.eyelid_color`
- Added validation behavior for malformed eyelid values:
  - API error id: `AUTORIG_EYELID_COLOR_INVALID`
- Added Blender + CLI user-facing controls for eyelid colors and opacity.
- Added runtime metadata emission:
  - `metadata.extensions.film.facial_plugin.appearance.eyelid_color`
